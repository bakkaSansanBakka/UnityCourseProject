Теоретические сведения

Гидроксокарбонат меди(II) (малахит) — неорганическое соединение, основная соль металла меди и угольной кислоты с формулой (CuOH)2CO3, тёмно-зелёные кристаллы, нерастворимые в воде.

Получение малахита:

  • Природа
  • Медь во влажном воздухе медленно реагирует с кислородом и диоксидом углерода:

    2Cu + CO2 + O2 + H2O → (CuOH)2CO3↓

  • Пропускание углекислоты через суспензию гидроксида меди:

    2(CuOH)2 + CO2 → (CuOH)2CO3↓ + H2O

  • Реакция сульфата меди и гидрокарбоната натрия:

    2CuSO4 + 4NaHCO3 → (CuOH)2CO3↓ + 2Na2SO4 + 3CO2↑ + H2O

Химические свойства:

  • Разлагается при нагревании:

    (CuOH)2CO3 → 2CuO + CO2 + H2O

  • Реагирует с кислотами:

    (CuOH)2CO3 + 4HCl → 2CuCl2 + CO2↑ + 3H2O

    (CuOH)2CO3 + 4CH3COOH → 2Cu(CH3COOH)2 + CO2↑ + 3H2O

  • Реагирует с солями аммония и раствором аммиака:

    (CuOH)2CO3 + 4NH4Cl → 2CuCl2 + CO2↑ + 4NH3↑ + 3H2O

    (CuOH)2CO3 + 8(NH3•H2O) → [Cu(NH3)4]CO3 + [Cu(NH3)4](OH)2 + 8H2O

  • Реагирует с цианистым калием:

    (CuOH)2CO3 + 8KCN → 2K2[Cu(CN)4] + K2CO3 + 2KOH

  • Реагирует с оксидом углерода (IV) под давлением:

    (CuOH)2CO3 + CO2 → 2CuCO3 + H2O

  • Растворяется в избытке горячего концентрированного раствора карбоната натрия с образованием синего раствора, при охлаждении из которого кристаллизуется вегшайдерит, окрашенный в синий цвет тетрагидроксокупратом натрия:

    (CuOH)2CO3 + 7Na2CO3 + 6H2O → 2Na5[CO3](HCO3)3 + 2Na2[Cu(OH)4]

В данном симуляторе будет проводиться опыт по разложению малахита при нагревании.

Для этого будет смоделирована нагревательная установка, состоящая из лабораторного штатива с зажимом, пробирок, спиртовки, газоотводной трубки, штатива для пробирок.

Зажим для штатива будет держать пробирку над спиртовкой, пока вещество (малахит) в ней будет нагреваться. Газоотводная трубка одним концом будет находится в нагреваемой пробирке, а другим во второй пробирке, в которой находится вещество, реагирующее с выделяемым при нагревании газом. Таким образом будет доказываться, что газ - CO2 (углекислый). После полного разложения малахита в пробирку с образовавшимся веществом добавляется раствор серной кислоты, после чего раствор окрашивается синим цветом - индикатор того, что в растворе содержится медь.
